# express-twitter-sample
Node+ExpressでTwitterでAuthするサンプルプログラムです


## usage
create twitter apps, write ./config.js
https://apps.twitter.com/app/new
```
exports.consumerKey = '';
exports.consumerSecret = '';
exports.callbackURL = '';
```


```
npm install
npm start
# start http server at localhost:3000
```
